# TextEditor

### Project Overview

A Simple text editor for day-to-day use.

### Features
- Provides basic text manipulation
- Status bar with live word & line count
- Interactive UI

### Tech Stack
- Lanugage: Python
- Libraries: CustomTkinter, Pillow

### Project Structure
- `main.py`: The main controller containing function to start the program
- `requirements.txt`: File containing all the required python libreries
- `Assets`
    - `Data`: Contains text files that contains the user settings since a database is not connected
    - `Images`: Contains the images used by the application
    - `PythonFiles`
        - `Variable.py`: Contains all the variables of the application
        - `Widgets.py`: Contains UI elements and logic of the application


### Getting Started

#### Prerequisites
- Make sure you have Python installed. You can check by running, 
```
python --vesion
```

#### Installation
- Clone the project to your system
    ```
    git clone
    cd TextEditor
    ```
- Then install all the requirements
    ```
    pip install -r requirement.txt
    ```

#### Running the App
- Run,
```
python main.py
```
